# 📊 Customer Behavior Analysis | End-to-End Data Analytics Project

## 📌 Overview

This project demonstrates a complete Data Analytics workflow using Python, PostgreSQL, SQL, and Power BI. The objective is to analyze customer behavior, clean and transform the data, extract business insights through SQL queries, and build an interactive dashboard for decision-making.

The project follows an end-to-end analytics pipeline, from raw data to business reporting.

---

## 📂 Dataset

- **Dataset:** Customer Shopping Behavior Dataset
- **Source:** Kaggle
- **Records:** 1,000+ customer transactions
- **Features Include:**
  - Customer ID
  - Age
  - Gender
  - Category
  - Purchase Amount
  -  - Payment Method
  - Shipping Type
  - Subscription Status
  - Review Rating
  - Frequency of Purchase
  - Discount Applied
  - Promo Code Used

## 🚀 Project Workflow

### 1. Data Loading
- Imported the CSV dataset into Python.
- Loaded data into a Pandas DataFrame.

### 2. Exploratory Data Analysis (EDA)
- Checked dataset dimensions.
- Explored data types.
- Identified missing values.
- Generated summary statistics.
- Analyzed customer demographics and purchasing patterns.

### 3. Data Cleaning & Preprocessing
- Removed unnecessary columns.
- Handled missing values.
- Converted categorical values where required.
- Created new features for better analysis.
- Prepared the dataset for SQL and Power BI.

- ### 4. PostgreSQL & SQL Analysis
- Connected Python to PostgreSQL using SQLAlchemy.
- Imported the cleaned dataset into PostgreSQL.
- Executed SQL queries to answer business questions.
- Performed aggregations, filtering, grouping, and sorting.

### 5. Power BI Dashboard
Developed an interactive dashboard containing:

- Total Customers
- Average Purchase Amount
- Average Review Rating
- Revenue by Category
- Sales by Category
- Revenue by Age Group
- Sales by Age Group
- - Customer Subscription Analysis
- Interactive Filters (Gender, Category, Subscription Status)

### 6. Reporting
- Documented key business insights.
- Created a professional project report.
- Designed a presentation (PPT) with AI assistance.

## ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/yourusername/customer-behavior-analysis.git
```

### Install required libraries

```bash
pip install pandas numpy matplotlib sqlalchemy psycopg2
```

### Run the project

1. Open the Jupyter Notebook.
2. Execute the notebook cells sequentially.
3. Connect Python to your PostgreSQL server.
4. Load the cleaned dataset into PostgreSQL.
5. Run the SQL queries.
6. Open the Power BI (.pbix) file to explore the dashboard.
